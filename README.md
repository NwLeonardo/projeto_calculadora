# Projeto Calculadora

## Descrição
Este projeto consiste no desenvolvimento do motor de cálculos para o aplicativo de finanças pessoais. O sistema realiza operações matemáticas fundamentais com foco em robustez, tratamento de exceções e documentação técnica.

## Objetivo da Atividade
Desenvolver competências práticas em:
- Testes Unitários e Funcionais;
- Tratamento de Erros (Exceções);
- Refatoração de Código;
- Documentação com JavaDoc;
- Versionamento com Git/GitHub.

## Tecnologias Utilizadas
- **Linguagem:** Java
- **Gerenciador de Dependências:** Maven
- **IDE:** Apache NetBeans
- **Documentação:** JavaDoc

## Operações
A classe `Calculadora` suporta:
1. **Adição (+)**
2. **Subtração (-)**
3. **Multiplicação (*)**
4. **Divisão (/)** - Com bloqueio de divisão por zero.

## Execução dos Testes
Os testes validam cenários de sucesso e o tratamento de entradas inválidas.
